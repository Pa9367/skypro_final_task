Git

  • git add - дабавление файлов в индекс;
  • git status - проверка статуса репозитория;
  • git commit - добавление файлов в репозиторий
  • git pull - работает как комбинация команд "git fetch" и "git merge": Git в начале забирает изменения из указанного удалённого репозитория, а затем пытается слить их с текущей веткой
  • git push - используется для установления связи с удалённым репозиторием,вычисления локальных изменений,отсутствующих в нём,и их передачи в репозиторий.
